Filtering Rules for Ontario North American Birds Regional Reports

This static page provides a graphical representation of the rules used to filter bird reports when genertating the Ontario regional reports for North American Birds. A list of all uploaded reports for Ontario can be found here. Click on the name of the species to see a table of the start date, end date, and geo bounding box (if applicable) for when this species is notable and should be included in the report. If there is more than one rule for a species, an observation of that species matching any one of the rules merits inclusion. To view the code used to filter sightings from eBird and iNaturalist, see NAB dataset filter. This rules list is always evolving and any input on adjusting date ranges or geo bounding boxes is always appreciated. For more information on contributing a rules change, see the contributing section of the readme.
